# Book-Tracker
## Автор
Обучающийся

## Описание
Приложение для учета прочитанных книг.
Можно добавлять книги, фильтровать их и сохранять данные в JSON.

## Функции
- Добавление книг
- Фильтрация по жанру и страницам
- Сохранение и загрузка данных

## Запуск
1. Установить Python
2. Запустить:
   python app.py
   
## Пример использования
Добавьте книгу → она появится в списке → используйте фильтр
